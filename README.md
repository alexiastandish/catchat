## Table of Contents

- [General Info](#general-info)

## General Info

Cat Chat is a social media platform about cats and provides the ability for users to chat live with one another

## Technologies

- Javascript
- React.js
- Socket.io
- Express
- Node.js
- PostgreSQL
- Google Firebase
- SASS

## Setup

Use the package manager npm to install store

```
$ cd ../catchat
$ npm install
$ npm start
```
